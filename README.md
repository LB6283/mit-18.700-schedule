# mit-18.700-schedule
I DID NOT MAKE THIS SCHEDULE! This was made by my friend who is a math major. I decided to post it on GitHub becuase I think it is super cool and I love math.
This is the schedule for the MIT OpenCourseWare 18.700 Linear Algebra course. This is an undergraduate-level course, and you can view the website here: 
https://ocw.mit.edu/courses/18-700-linear-algebra-fall-2013/ 

Here is the description of the course provided on the website:

"This course offers a rigorous treatment of linear algebra, including vector spaces, systems of linear equations, bases, linear independence, matrices, determinants, eigenvalues, inner products, quadratic forms, and canonical forms of matrices. Compared with 18.06 Linear Algebra, more emphasis is placed on theory and proofs."

The PDF was compiled from LaTeX code in Overleaf, and contains links to notes, chapters, and assignments. You can also compile the PDF yourself from the latex code provided.
I know it works on the following operating systems:
 - Fedora Linux (Probably most Linux Distros)

My friend built this on MacOS, but the file structure may have been different on his machine. This will probably work on MacOS but I'm not 100% sure.

Instructions:

1. Download the folder titled 18.700
2. Navigate to 18.700/Schedule
3. Open the PDF
4. There are clickable links in the PDF that will take you to the assignment.
